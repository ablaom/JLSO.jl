    Manually reading JLSO files can be helpful when addressing issues deserializing objects or to simply to help with reproducibility.

    julia> jlso = read("breakfast.jlso", JLSOFile)
JLSOFile([cost, food, time]; version="3.0.0", julia="1.0.3", format=:julia_serialize, compression=:gzip, image="")

    These project and manifest fields are just the dictionary representations of the Project.toml and Manifest.toml files found in a Julia Pkg environment. In the future, we may add some tooling to make it easier to view and compare these dictionaries, but it's currently unclear if that should live here or in Pkg.jl.
